Instrucciones de uso:

1. La Ventana de Captura de Huellas servira para Agregar y/o Editar las Huellas.
2. Al abrirse la ventana se mostrara una de las siguientes leyendas:
     I. "Dispositivo Encontrado" significa que se podra trabajar para la captura de huellas
        sin problema, y se activara automaticamente el Boton de "Guardar".
    II. "Dispositivo NO Encontrado" significa que el Lector de Huellas no se encuentra
        conectado o si esta conectado no fue detectado por el sistema y se debe de cerrar
        y abrir la ventana de captura de Huellas de nuevo. El Boton de "Guardar" quedara
        desactivado para no permitir ningun cambio.
3. En la Imagen donde se muestran los 10 dedos se podra diferenciar que dedos fueron ya
   capturados y cuales todavia les falta realizar la captura, donde:
     I. "Amarillo" al Dedo le falta capturar la Huella.
    II. "Verde"    el Lector de Huellas esta listo para capturar el Dedo seleccionado.
   III. "Azul"     el Dedo ya fue capturado pero pendiente por guardar.
    IV. "Blanco"   el Dedo ya se encuentra guardado en la Base de Datos.
4. Para poder activar el proceso de lectura de Huellas se debe dar un clic sobre el numero
   del dedo que se va a capturar o pulsando las teclas F1 a F10 segun el dedo a capturar, por
   ejemplo, si se captura el Dedo No. 2 se dara clic en el recuadro que contiene el 2 o en
   su defecto precionar la tecla F2 y se podra proceder a la captura de la Huella.
5. La captura de Huellas inicia cuando el numero del Dedo tenga de fondo el color: "Verde".
6. Al estar capturando una Huella (numero con fondo "Verde") se debera poner la misma Huella
   3 veces por razones de verificacion, se podra identificar cada captura por medio de la
   Barra de Estado ubicada debajo de la imagén de las manos, cuando se realize correctamente
   se cambiara a "Azul" el fondo del numero que se esta capturando, sino se manda un mensaje
   al usuario indicandole que no se pudo completar la Accion y se debe de intentar de nuevo.
7. Si se desea cancelar la captura de un Dedo (cuando este con fondo "Verde") se dara clic
   sobre ese mismo numero dejandole su color que tenia por default:
     I. "Amarillo" si no se ha capturado todavia una Huella para ese Dedo.
    II. "Azul"     si ya se habia capturado una Huella para ese Dedo.
   III. "Blanco"   si ya se habia capturado una Huella para ese Dedo guardado en la BD.
8. No existe ningun orden en la captura de Huellas y pueden editarse.
9. Si se realizaron cambios y se trata de cerrar la ventana se le preguntara al usuario si
   aun asi desea salir recordandole sobre los cambios.
